# Student Management System - Assignment 1

## A. Project Overview
This project is an Object-Oriented Student Management System designed to manage student records and course data. It applies Java fundamentals like loops and arrays while implementing core OOP principles like encapsulation and composition.

## B. Class Descriptions
* **Student Class**: Represents a student with private fields for name, ID, major, GPA, and credits. It uses getters and setters for data hiding and proper state management.
* **Course Class**: Demonstrates composition by storing an array of Student objects. It provides functionality to calculate the average GPA and identifies the student with the highest credits.
